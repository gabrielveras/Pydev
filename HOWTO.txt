# Git:
## Download and install Git (Any version, last used 2.18.0)
https://git-scm.com/downloads
## Download PyDev source-code
1) Open the console in the desired folder and use the follwing commands.
> git clone https://github.com/gabrielveras/Pydev.git
> cd PyDev
> git checkout PySwarm
> git pull
> git fetch

# Eclipse SDK
## Download Eclipse SDK (Version 4.8.0)
http://download.eclipse.org/eclipse/downloads/drops4/R-4.8-201806110500/
## Opening project
1) Create new folder for Pydev workspace
2) Open eclipse and assign this folder as workspace
3) Go to top menu > File > Import.
4) Select "Existing Projects into Workspace" option then click on Next button.
5) Browse for the source code folder then click on Finish button.
6) Run as Eclipse Application
